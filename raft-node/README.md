# raft-node

> Raft node implementation — types, protocol logic, and swappable components

`raft-node` provides the single-node Raft implementation under the 3-6 model, including `RaftNode<P>`, `RaftProtocol<P>`, in-memory implementations, builders, and protocol-level tests.
