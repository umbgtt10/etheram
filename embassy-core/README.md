# embassy-core

> Shared `no_std` Embassy infrastructure for protocol-family embassy crates

`embassy-core` contains protocol-agnostic embedded infrastructure reused by `etheram-embassy` and `raft-embassy`: channel transport hubs, timer channels, network bus/config helpers, outbox transport bridge, and external-interface/client-facade macros.

**Parent:** [EtheRAM](../README.md)
**Depends on:** [core](../core/README.md)
**Used by:** [etheram-embassy](../etheram-embassy/README.md), [raft-embassy](../raft-embassy/README.md)
